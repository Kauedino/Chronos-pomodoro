# ⏳ Chronos Pomodoro

O **Chronos Pomodoro** é uma aplicação web desenvolvida para ajudar usuários a gerenciarem seu tempo e aumentarem sua produtividade utilizando a técnica Pomodoro. Este projeto foi criado durante o meu aprendizado de React avançado.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as ferramentas mais modernas do ecossistema de desenvolvimento web:

* **React** (Biblioteca principal para construção da interface)
* **TypeScript** (Tipagem estática para maior segurança e produtividade)
* **Vite** (Ferramenta de build ultra-rápida)
* **Git & GitHub** (Controle de versão e hospedagem do código)

## 💻 Como rodar o projeto localmente

Se você quiser baixar este projeto na sua máquina para testar ou estudar, siga os passos abaixo no seu terminal:

1. **Clone o repositório:**
```bash
   git clone git@github.com:Kauedino/Chronos-pomodoro.git

    Entre na pasta do projeto:

Bash

   cd Chronos-pomodoro

    Instale as dependências:

Bash

   npm install

    Inicie o servidor de desenvolvimento:

Bash

   npm run dev

    Agora é só abrir o navegador no endereço indicado no terminal (geralmente http://localhost:5173).
